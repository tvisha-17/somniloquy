  <behavior>
    - `map_sleep_stages(annotations) -> np.ndarray` returns int array with the mapping: values containing "W"/"wake"→0, "N1"/"1"→1, "N2"/"2"→2, "N3"/"3"→3, "R"/"REM"/"4"→4, anything else→-1. Case-insensitive.
    - `bandpass_notch_resample(raw, l_freq, h_freq, notch_freqs, target_sfreq) -> mne.io.Raw` applies filter → notch → resample in that order and returns the modified raw.
    - `epoch_data(raw, sleep_stages, epoch_duration, overlap) -> Tuple[np.ndarray, np.ndarray, np.ndarray]` returns `(data, stages, start_times)` where `data` has shape `(n_epochs, n_channels, n_timepoints)` with `n_timepoints = int(epoch_duration * raw.info["sfreq"])`.
    - `reject_by_peak_to_peak(data, stages, start_times, threshold_v) -> Tuple[np.ndarray, np.ndarray, np.ndarray]` drops epochs where `(max - min)` on any channel exceeds `threshold_v`.
    - `zscore_per_channel(data) -> np.ndarray` normalizes each channel across all epochs so the per-channel mean is ~0 and std is ~1. Input/output dtype is float32.
    - `preprocess_subject(raw_path: Path, cfg: dict) -> dict | None` returns `None` and logs a warning when final epoch count < `cfg["min_epochs"]`, otherwise returns `{"data", "sleep_stages", "subject_id", "sfreq", "ch_names", "epoch_times_s"}`.
    - `save_subject_npz(subject_dict: dict, output_dir: Path) -> Path` writes `sub-<id>_epochs.npz` with all keys and returns the path.
    - Output validation inside `preprocess_subject`: asserts dtype float32, no NaN, `abs(data.mean()) < 0.1`, `abs(data.std() - 1.0) < 0.2`.
  </behavior>
  <action>
    Create `src/data/preprocess_dream.py` implementing all functions above. Use MNE for filter/notch/resample:
    ```python
    raw.pick_types(eeg=True)
    raw.filter(l_freq=l_freq, h_freq=h_freq, fir_design="firwin", verbose=False)
    raw.notch_filter(freqs=notch_freqs, verbose=False)
    raw.resample(target_sfreq, verbose=False)
    ```

    For epoching, create fixed-length windows using `mne.make_fixed_length_epochs(raw, duration=epoch_duration, overlap=epoch_duration*overlap, preload=True)` and get the data via `epochs.get_data()`. Align sleep stages by taking the annotation active at each epoch start time; implement a helper `_stage_at_time(annotations, t)` that scans `raw.annotations` and returns the mapped code (or -1 if none covers `t`).

    Rejection: compute `ptp = data.max(axis=-1) - data.min(axis=-1)` → shape `(n_epochs, n_channels)`. Keep epochs where `ptp.max(axis=-1) <= threshold_v`.

    Normalization: `mean = data.mean(axis=(0, 2), keepdims=True)`, `std = data.std(axis=(0, 2), keepdims=True) + 1e-8`, `data = ((data - mean) / std).astype(np.float32)`.

    `preprocess_subject` extracts subject ID from filename via regex `r"sub-([A-Za-z0-9]+)"`. Log at INFO using `get_logger(__name__)` at each stage: "loaded shape=...", "post-filter shape=...", "post-epoch shape=...", "post-reject n_epochs=...", "post-normalize mean=...,std=...".

    `save_subject_npz` uses `np.savez(path, data=..., sleep_stages=..., subject_id=..., sfreq=..., ch_names=np.array(ch_names), epoch_times_s=...)`.

    Create `tests/test_preprocess_dream.py` with these tests (use synthetic MNE raw via `mne.io.RawArray` with known data, no real files):
    1. `test_map_sleep_stages_known_codes()` — asserts `map_sleep_stages(["Wake","N1","N2","N3","REM","foo"])` returns `[0,1,2,3,4,-1]`
    2. `test_map_sleep_stages_case_insensitive()` — asserts lowercase input works
    3. `test_zscore_per_channel_produces_unit_variance()` — input `(10, 4, 100)` random float32, after zscore assert `abs(out.mean()) < 0.1` and `abs(out.std() - 1.0) < 0.2` and `out.dtype == np.float32`
    4. `test_reject_by_peak_to_peak_drops_high_ptp()` — build data where epoch 0 has ptp=500e-6 and the rest have ptp=50e-6, threshold=200e-6, assert the first epoch is dropped and stages/start_times are aligned
    5. `test_epoch_data_shape_correct()` — construct a `mne.io.RawArray` with sfreq=256, 4 channels, 20 seconds of data; epoch at 2s/0 overlap, assert returned shape is `(10, 4, 512)`
    6. `test_preprocess_subject_skips_if_below_min_epochs(monkeypatch, tmp_path)` — monkeypatch `_load_raw` to return a short raw yielding < `min_epochs` epochs, assert return value is `None`
    7. `test_save_subject_npz_roundtrip(tmp_path)` — save a minimal dict, reload with `np.load`, assert all keys present and `data.dtype == np.float32`
    8. `test_output_validation_passes_on_normalized(tmp_path)` — run `preprocess_subject` on a synthetic raw with enough duration, assert returned dict satisfies `data.dtype==float32`, no NaN, `abs(mean)<0.1`, `abs(std-1)<0.2`
